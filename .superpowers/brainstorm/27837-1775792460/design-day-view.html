视图切换

原有"文本/JSON"切换保留。新增"按天"Tab。切换后进入按天视图。

按天聚合

天区块

每天一个区块，显示当天有更新的所有任务（完成 ✓ 或进度中 …）。

拖拽

工时调整

每行右侧有一个拖拽进度条，左右拖动调整当天工时（0.5h 步进）。数字实时更新。

汇总

每天总计

天标题右侧实时显示当天所有任务工时之和。超过 8h 时变红提示。

复制

生成 JSON

点击"复制"，用户拖拽调整的工时作为各任务的 plannedHours；duration 固定为当天 [dayStart, dayEnd]。

工时拖拽交互

在任务行右侧，有一个小进度条 + 数值。
鼠标左右拖动进度条，数值以 0.5h 为步进实时变化。

说明

任务跨天展示

若一项任务在周一和周二都有记录（完成或进度更新），则在两天各自独立出现，工时单独设置。

注意

超时提示

当天总工时 > 8h 时，右上角合计数字变红（如周二 10h）。仅提示，不限制。

数据

复制 JSON 逻辑

每个任务在每天生成一条 JSON 记录：
· plannedHours = 拖拽设定的工时
· duration = [当天00:00, 当天23:59]
· hours = 同 plannedHours
